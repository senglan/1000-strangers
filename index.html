How this works

One person built this page and gave it a 24-hour window. The count you see is from GoatCounter, an open-source, privacy-friendly counter: no cookies, no personal data, nothing to opt out of. Known bots, crawlers, and link-preview robots are filtered out — the number is humans (as best the internet allows).

Every verified visitor reveals one cell of the hidden picture, in an order fixed before launch. Your visitor number is stored only in your own browser, so the page remembers which pixel is yours.

The rules of the experiment: no paid promotion, no link aggregators, no begging algorithms. The page moves through group chats or it doesn't move at all.
